# daughtertalks 女儿的谈话

我女儿名叫Grace,3岁了，她经常说些奇怪的话，有必要记下来，很有趣。
对于一个软件程序员来讲，可能最好和简单的存放地方是在github开一个新项目...

2018/3/15 别当猴子了
<br />故事背景: 一家人在动物角色扮演游戏后，晚上在床上准备睡觉。
<br />女儿突然想到一件事情，对爸爸说：爸爸，你以后别当猴子了。
<br />爸爸：为什么呢？
<br />女儿：你太大了，会把树压坏的。你别当猴子了。
